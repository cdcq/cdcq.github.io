西电校赛2022H题——朝歌夜雪

发表于2022-05-25更新于2022-05-25阅读次数

题意：

给你 n 个变量 c1,c2,...,cnc_1, c_2, ..., c_nc1​,c2​,...,cn​ ，现在让你往里放任意整数，要求

  1. ci−1∣cic_{i-1}|c_ici−1​∣ci​
  2. 1≤ci≤C1 \leq c_i \leq C1≤ci​≤C

给你 nnn 和 CCC ，问你有多少种放数的方案满足上述要求。答案模 998244353 。

这道题现场没敢写，首先感觉会不会是数论函数之类的题，那种我做不来。然后简单想了一下感觉像是组合数 dp ，这种我也不擅长，以前翻车过很多次，就直接弃疗了。考完看题解才发现其实我想得已经很接近了，只需要再坚持一下就能做出来。果然 ACM 最后一个小时一定不能松懈，坚持就是胜利啊 233 。

现场我想的时候总是拘泥于 C 的限制，所以不好做。也想过枚举最后一个数，但是考虑到 n 是 1e5 还是没继续想。其实枚举最后一个数之后复杂度就跟 n 无关了，可以直接公式算。

假设最后一个数为 p1q1p2q2...pkqkp_1^{q_1}p_2^{q_2}...p_k^{q_k}p1q1​​p2q2​​...pkqk​​ ，由于不同质因子之间完全独立的，所以可以一次只考虑一个质数幂因子，最后用乘法原理乘起来。

那么对于 piqip_i^{q_i}piqi​​ ，方案数其实就是把 qiq_iqi​ 个质因子分配到 n 个变量里的问题。用插板法，答案就是 C(n−1,qi+n−1)C(n-1, q_i+n-1)C(n−1,qi​+n−1) 。

计算组合数可以预处理 1 到 n+mn+mn+m 的阶乘和阶乘的逆元，然后直接用组合数的阶乘公式算。复杂度是 O(CC)O(C\sqrt C)O(CC​) 。

注意不要测完 100000 100000 就 OK 了，记得手测边界情况（如 1 100000 ）。
